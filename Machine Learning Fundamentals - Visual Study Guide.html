lightbulb Introduction to Machine Learning

Machine learning is a subset of AI focused on algorithms that learn patterns from training data and make accurate inferences about new data.

Key components:

  • Training data: The dataset used to train the model
  • Algorithm: The method used to learn patterns
  • Model: The output of the training process
  • Predictions: The model's output on new data

Example: A Twitter program that learns to filter bullying tweets based on user labeling.

category Types of Machine Learning

Types of Machine Learning

  • Supervised Learning: Uses labeled data for training
    • Classification Predicts discrete categories (spam/not spam)
    • Regression Predicts continuous values (house prices)
  • Unsupervised Learning: Finds patterns in unlabeled data
    • Clustering Groups similar data points
  • Semi-Supervised Learning: Combines labeled and unlabeled data
  • Self-Supervised Learning: Creates labels from data itself
  • Reinforcement Learning: Learns through interaction with environment

trending_up Linear Regression

Linear Regression vs Logistic Regression

Simple Linear Regression: One independent variable

Y = mX + b
  • Best fit line minimizes squared errors
  • Cost function: Mean Squared Error (MSE)

Multiple Linear Regression: Multiple independent variables

Y = b₀ + b₁X₁ + b₂X₂ + ... + bₙXₙ

Prediction vs Explanation:

  • Prediction: Focus on accuracy with unseen data
  • Explanation: Focus on understanding relationships

compare_arrows Logistic Regression

Sigmoid Function in Logistic Regression

Predicts probability between 0 and 1

p = 1/(1+e⁻ᶻ)

Decision boundary: p ≥ 0.5 → Class 1, p < 0.5 → Class 0

Cost function: Log loss (cross-entropy)

Evaluation metrics: Accuracy, Precision, Recall, F1 Score

account_tree Decision Trees

Flowchart-like structure for decision making

Splitting criteria:

  • Gini Impurity: Measures node impurity
  • Entropy: Measures information gain
  • Information Gain: Entropy(parent) - Weighted Entropy(children)

Recursive partitioning until stopping criteria met

Overfitting prevention: Maximum depth, minimum node size

functions Naive Bayes

Based on Bayes' Theorem:

P(y|X) = P(X|y)P(y)/P(X)

"Naive" assumption: Features are conditionally independent

Types:

  • Gaussian: Continuous values
  • Multinomial: Frequency of terms
  • Bernoulli: Binary features

Works well for text classification and spam filtering

share K-Nearest Neighbors (KNN)

KNN Classification with Different K Values

Non-parametric, instance-based learning

Classification: Majority vote of k nearest neighbors

Regression: Average of k nearest neighbors

Distance metrics:

  • Euclidean: Straight-line distance
  • Manhattan: Sum of absolute differences

Choosing k: Small k → sensitive to noise, Large k → smoother boundary

Feature scaling is crucial (Z-score normalization)

Central Limit Theorem Visualization

Random Sampling vs Sample Bias

Central Limit Theorem: Distribution of sample means approximates normal

SE = s/√n

Bootstrap: Resampling with replacement to estimate sampling distribution

Confidence Intervals: Range likely to contain true population parameter

view_comfy Ensemble Methods

Decision Tree vs Random Forest

Bagging (Bootstrap Aggregation):

  • Creates multiple datasets using bootstrapping
  • Trains separate models on each dataset
  • Combines predictions (average or majority vote)

Random Forest:

  • Bagging + Random feature selection
  • Each tree selects random subset of features
  • Reduces variance and improves accuracy
  • Variable importance measurement
