Web Engineering Viva Study Notes

language Introduction to Web Structure

Hypertext - Text with hyperlinks that connect to other content
HTTP - Protocol for transferring hypertext requests and information
IP Address - Unique numerical label assigned to devices
IPv4 vs IPv6 - IPv4: 32-bit addresses (e.g., 192.168.1.1), IPv6: 128-bit addresses
Website vs Web Application - Website: informational, Web App: interactive functionality
Domain Name Structure - Top-level, second-level, subdomain structure
Web Request-Response Cycle - Browser sends request, server processes and returns response
Remember: HTTP is stateless - each request is independent!

style Introduction to CSS

CSS Documents & The Cascade - Multiple stylesheets cascade with specificity rules
Selectors, Properties & Values - Target elements, apply styles with specific values
Classes & IDs - Class: reusable, ID: unique element identifier
Specificity in CSS - ID > Class > Element > Universal selector
Flex Box - One-dimensional layout model for arranging elements
Length Units - px (pixels), % (percentage), rem (root em), em (relative), vh (viewport height)
The Box Model - Content, Padding, Border, Margin
Position Property - static, relative, absolute, fixed, sticky
CSS Pseudo Classes - :hover, :active, :focus, :first-child, etc.
Flexbox is perfect for responsive layouts!

code Introduction to JavaScript

Variables & Const Variables - let (reassignable), const (immutable), var (function-scoped)
Numbers & Strings - Primitive data types for numeric and text values
Booleans & Type Conversion - true/false values, implicit/explicit type conversion
Arrays - Ordered collection of values, zero-indexed
Objects - Key-value pairs for storing structured data
Operators - Arithmetic, Relational, Increment & Decrement
Conditional Statements - if, else-if, switch with logical operators (&&, ||)
Loops - for, while, do-while, for...of, for...in
Functions - Reusable blocks of code, can take parameters and return values
Arrow functions (=>) have different 'this' binding than regular functions!

account_tree Introduction to DOM

DOM (Document Object Model) - Programming interface for HTML documents
querySelector - Returns first element matching CSS selector
querySelectorAll - Returns all elements matching CSS selector
getElementById - Returns element with specific ID
getElementsByTagName - Returns collection of elements with specific tag name
getElementsByClassName - Returns collection of elements with specific class
DOM Manipulation - Changing content, structure, and style of documents
Changing Attributes & Values - element.setAttribute(), element.value
Add, Create & Remove Elements - createElement(), appendChild(), removeChild()
querySelector is more versatile but getElementById is faster!

html HTML Manipulation

Getting & Setting Values - element.value, element.textContent, element.innerHTML
Handling Attributes - getAttribute(), setAttribute(), removeAttribute()
Inserting/Removing Elements - insertBefore(), appendChild(), remove()
CSS Manipulations - element.style, element.classList
innerHTML can be a security risk with user input - use textContent instead!

upgrade ES6 Features

Arrow Function - Concise syntax: (param) => { return value; }
Template Literal - String interpolation with backticks: `Hello ${name}`
ES6 (ECMAScript 2015) introduced many modern JavaScript features!

react Introduction to React.js

Core Features - Component-based, Virtual DOM, Unidirectional data flow
Thinking in React.js - Break UI into components, build static version, add state
Building App Using Components - Reusable UI pieces with props and state
Creating Custom Components - Functional components with hooks or class components
Create Hello World App - Basic React app structure with JSX
Routing in React.js - BrowserRouter, Routes, Route for navigation
React uses JSX - JavaScript syntax extension that looks like HTML!

dns Introduction to Node.js

Node.js - JavaScript runtime built on Chrome's V8 engine
Setup Development Environment - Install Node.js, choose code editor
Node.js Console - REPL (Read-Eval-Print-Loop) for testing code
Node Package Manager (NPM) - Package manager for Node.js modules
Installing Packages Locally - npm install package-name
Node.js allows JavaScript to run outside the browser!
